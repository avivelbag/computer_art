What is a Fourier Series?

Any periodic signal — no matter how complex — can be written as an infinite sum of simple sine and cosine waves. This is Fourier's theorem (1822), one of the most powerful ideas in mathematics and engineering.

Each term in the sum is called a harmonic. The first harmonic (fundamental) oscillates once per period; the n-th harmonic oscillates n times per period.

Square Wave

A perfect square wave uses only odd harmonics, each with amplitude 1/n:

f(x) = 4/π · Σ sin((2k−1)x) / (2k−1)
k = 1, 2, 3, …

Only odd harmonics appear because the square wave has half-wave symmetry: f(x + π) = −f(x). Even harmonics would break that symmetry.

Sawtooth Wave

A sawtooth uses all harmonics with amplitudes 1/n and alternating signs:

f(x) = 2/π · Σ (−1)^(k+1) · sin(kx) / k
k = 1, 2, 3, …

Triangle Wave

A triangle wave uses odd harmonics like the square wave, but amplitudes fall off faster (1/n²), giving smoother corners:

f(x) = 8/π² · Σ (−1)^k · sin((2k+1)x) / (2k+1)²
k = 0, 1, 2, …

Phasor View

Each active harmonic is drawn as a rotating arrow (phasor) on the left. The n-th harmonic rotates n times faster than the fundamental. The tip of the final arrow traces the output signal, which is simultaneously drawn on the right panel.
