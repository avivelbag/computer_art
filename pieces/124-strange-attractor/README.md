# Piece 124 — Strange Attractor: A Basin with No Bottom

A Clifford strange attractor iterated for 5 million steps and rendered through
additive per-pixel accumulation.  The orbit is nowhere periodic, yet it is
forever bounded — tracing the same ghostly shape without ever exactly repeating.

## Attractor equations

The Clifford attractor is a 2-D discrete-time map:

```
x' = sin(a·y) + c·cos(a·x)
y' = sin(b·x) + d·cos(b·y)
```

Parameters used here:

| parameter | value |
|-----------|-------|
| a         | −1.4  |
| b         |  1.6  |
| c         |  1.0  |
| d         |  0.7  |

These values produce the characteristic butterfly / double-lobe shape whose
wings span roughly [−2.5, 2.5]² in phase space.

## Visual technique

The canvas is initialised to a deep-space violet (`#0d0820`).  On each
animation frame, 10 000 new orbit points are computed.  Each point is mapped to
an 800×800 pixel grid and written directly into an `ImageData` buffer using
additive blending — the existing pixel value is incremented by a small amount
rather than overwritten.  Dense regions of the attractor accumulate brightness;
sparse regions stay dark, revealing the fractal density structure of the strange
attractor.

Point colour is derived from the y-coordinate:

- **y near −2.5** → deep violet (`rgb(100, 0, 200)`)
- **y near 0** → sky blue (`rgb(0, 150, 255)`)
- **y near +2.5** → warm gold (`rgb(255, 210, 0)`)

This gradient makes the two lobes of the butterfly visually distinct.

## Animation

At 10 000 points per animation frame the orbit reaches 5 million total
iterates in roughly 8 seconds at 60 fps, then the iteration counter resets to
zero and the density continues building on the same canvas.  After 60 seconds
the entire canvas clears and the cycle begins again from a fresh dark
background, so the gradual emergence of the attractor shape repeats.

## Thumbnail

`thumbnail.svg` is generated by `generate_thumbnail.py` using 2 000 orbit
samples (after a 500-step warm-up burn-in, every 5th point collected).  Each
sample is rendered as a filled circle with the same violet→sky-blue→gold colour
mapping, giving an immediate recognisable preview of the butterfly shape on a
480×480 canvas.
