# Frequencies in Common

A family of Lissajous curves rendered as a static SVG in the style of a pen plotter.

Each curve is defined by `x = sin(a·t + δ)`, `y = sin(b·t)` for a rational frequency ratio `a:b`. The composition samples 14 frequency pairs (from 1:2 through 5:7) and four phase offsets per pair, producing 56 curves that overlap and weave across an 800×800 canvas.

Thin strokes (0.5 px) at 25% opacity let overlapping curves build density gradients — the intersection regions accumulate ink the way a physical plotter would when traversing the same area multiple times.

**Technique:** SVG / Lissajous curves / parametric / pen-plotter  
**Medium:** Static SVG, no JavaScript  
**Generated by:** `generate.py` (Python stdlib only)
