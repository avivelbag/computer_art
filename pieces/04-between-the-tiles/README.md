# Between the Tiles

Each cell of a grid independently picks one of two Truchet quarter-circle arc orientations, producing a labyrinthine network of flowing curves from a purely binary choice. Every 80 ms a random tile flips orientation, so the pattern slowly ripples and evolves without ever settling into a fixed state.

The palette is strictly two colors — deep ink (`#1a1a2e`) and warm cream (`#f0e6d0`) — for a woodblock-print contrast that keeps the eye tracing the implied corridors.

**Technique:** HTML5 Canvas, Truchet tiles, quarter-circle arcs  
**Year:** 2026
