# Amber Current

600 particles drift through a Perlin-noise flow field in warm amber light.

Each particle follows the vector angle defined by 2D Perlin noise sampled at its
position. Trails fade slowly, leaving ghost-like streaks that reveal the
underlying current structure. The palette — deep amber, molten orange, pale gold
— recalls firelight through amber glass.

**Technique:** HTML5 Canvas, Perlin noise, flow-field particle simulation  
**Year:** 2026
