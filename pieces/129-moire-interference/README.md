# Piece 129 — Ghost Frequencies: A Moiré Study

Two systems of concentric rings share a canvas. The first is anchored at the center; the second drifts in a slow sinusoidal orbit — offset by up to 50 px horizontally and 30 px vertically. Where rings from both systems coincide, the canvas compositing mode `difference` cancels them to black, producing the interference fringe (the "ghost frequency"). As the offset evolves the beating dark band sweeps across the canvas like a standing wave, looping seamlessly because the motion is purely sinusoidal.

Palette is monochrome: near-black background (`#0d0d0d`), white ring strokes. No hue — only the geometry generates the visual event.
