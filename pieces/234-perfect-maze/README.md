# Perfect Maze: The Space That Carves Itself

A 27×27 perfect maze generated by the recursive backtracker (depth-first search) algorithm, rendered as a static SVG. The DFS carves passages by visiting every cell exactly once, guaranteeing a connected maze with no loops and a unique path between any two points. The amber solution path — found by BFS from top-left to bottom-right — traces the single shortest route through the labyrinth.

**Palette:** deep navy background (#0d0d1a), warm cream walls (#e8dcc8), amber solution path (#d4a853).

**Technique:** SVG / DFS recursive backtracker / BFS solver / static generation

The piece is a pivot from the original Truchet tile suggestion — Truchet is already represented in the gallery at pieces 04 and 65. Perfect maze generation is a genuinely distinct technique: the labyrinthine structure emerges from a single algorithm rather than per-tile randomness, producing long winding corridors and a clear visual tension between the ordered grid and the irregular passages carved through it.

## Regenerating

```bash
python3 gen.py
```

Outputs `piece.svg` (27×27 grid, 630×630 px canvas) and `thumbnail.svg` (15×15 grid, 238×238 px).
