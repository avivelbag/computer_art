# Quasicrystal — Five-Wave Interference

A quasicrystal interference pattern is generated by superimposing five cosine plane waves whose propagation directions are evenly spaced at 72° intervals: `f(x,y,t) = (1/5) Σᵢ cos(k·(x·cos(θᵢ+ωt) + y·sin(θᵢ+ωt)))`, where θᵢ = i·2π/5. Five-fold rotational symmetry is forbidden in periodic crystals but appears naturally here, producing a Penrose-tiling-like aperiodic structure. The pattern is continuously rotated (ω = 0.15 rad/s) by incrementing all wave-vector angles uniformly each frame, yielding a seamless 60 fps loop; field values near +1 (constructive interference) map to bright cyan-white (~`#7ef8e8`), values near −1 (destructive) collapse to near-black navy (`#020810`), with a dark teal midpoint — a 65536-entry cosine lookup table replaces Math.cos in the inner pixel loop to keep frame time well within the 16.7 ms budget.
